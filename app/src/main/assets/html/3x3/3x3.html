Элементы:

  • Центры

Кубик рубика имеет 6 центров, которые окрашены в разные цвета и определяют цвет грани. В процессе сборки кубика рубика их местоположение не меняется. В стандартном кубике рубике желтый центра располагается напротив желтого, синий - зеленый и оранжевый - красный.

  • Ребра

Состоят из двух цветов, которые нельзя разъединить между собой.

  • Углы

Состоят из трех цветов, которые нельзя разъединить между собой.

Язык вращения кубика рубика:

Так как на кубике всего 6 граней, для каждой из них используется свое обозначение:

R(right) — правая грань

L(left) — левая грань

U(up) — верхняя грань

D(down) — нижняя грань

F(front) — передняя грань

B(back) — задняя грань

Просто буква означает, что нужно сделать один поворот этой грани по часовой стрелке, буква со штрихом - один поворот против часовой стрелки, буква с двойкой - два поворота по часовой стрелки.
