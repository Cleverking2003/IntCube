Язык вращения кубика рубика:

Так как на кубике всего 6 граней, для каждой из них используется свое обозначение:

R(right) — поворот правой грани

L(left) — поворот левой грани

U(up) — поворот верхней грани

D(down) — поворот нижней грани

F(front) — поворот передней грани

B(back) — поворот задней грани

Просто буква означает, что нужно сделать один поворот этой грани по часовой стрелке, буква со штрихом - один поворот против часовой стрелки, буква с двойкой - два поворота по часовой стрелки.

  1. Сборка первого слоя

Собранный первый слой

Вначале нужно собрать первый слой на любой из сторон. Для удобства будем собирать его на белой. Находим на кубике любой угол с белым цветом. Например, бело-красно-зеленый. Этот уголок будет основой первого слоя.

Нужно разместить его внизу слева так, чтобы белая сторона смотрела вниз. Получается, что на нас смотрит зеленый цвет.

После этого находим еще один угол с белым цветом, у которого есть цвет, который смотрит на нас у первого угла. В нашем случае — это бело-зелено-оранжевый угол. Его нужно поставить перед нами сверху справа.

  1. Расстановка углов второго слоя

Пример правильно расположенных углов

На этом этапе нужно правильно расставить углы второго слоя. Чтобы цвета на каждом из верхних элементов, кроме желтого, совпадали с цветами элементов под ними.

Для этого нужно найти желто-красно-зеленый угол и поставить его на правильное место: над другим углом на первом слое с красным и зеленым цветом. Он может быть развернут по-разному, главное, чтобы он стоял на нужном месте.

Далее, не вращая грани кубика, осматриваем верхний слой и находим другие углы, которые стоят правильно. То есть цвета на них (кроме желтого) должны быть такими же, как цвета у углов, которые стоят под ними.

Если больше таких углов не нашлось, нужно взять любой другой угол на верхнем слое и поставить его на правильное место. Далее снова проверяем остальные уголки. Нужно найти ситуацию, когда хотя бы два уголка стоят на правильных местах.

Два правильно стоящих угла могут стоять рядом или по диагонали. Если они стоят рядом, нужно взять кубик так, чтобы правильные углы оказались слева, и выполнить алгоритм.
