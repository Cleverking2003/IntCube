IntCube - это мобильное приложение, позволяющее помочь пользователю решать головоломку Кубик Рубика. С помощью камеры мобильного телефона пользователь может отсканировать свой Кубик Рубика, а приложение определит тип кубика и поможет решить его. Также приложение предоставляет обучающие курсы по решению головоломки и возможность решать виртуальный кубик.

Установка и запуск:

Для установки приложения необходимо:

  1. Скачать apk-файл приложения с Google Диска
  2. Открыть apk-файл на мобильном устройстве
  3. Нажать на кнопку установки приложения

Инструкция по использованию:

Виртуальный кубик:

При нажатии кнопки "Виртуальный кубик" на главном экране приложения, пользователь может перейти на экран для взаимодействия с виртуальным кубиком.

Управление осуществляется при помощи касаний и движений пальцами. Если движение было начато за пределами кубика, то модель повернется, иначе будет повернута соответствующая грань.

Также на экране присутствуют четыре кнопки:

  1. 2х2, 3х3, Axis - переключают виртуальный кубик на 2х2, 3х3, Axis Cube соответственно.
  2. Reset - возвращает виртуальный кубик в исходное состояние.

Обучающие курсы:

При нажатии кнопки "Обучение" на главном экране приложения, пользователь переходит на экран с обучающими курсами. Каждый курс - отдельная кнопка с подписью, соответствующей типу головоломки, при нажатии на которую открывается страница, содержащая текст с инструкциями по сборке.

Сканирование:

  1. На главной странице нажать кнопку "Сканирование"
  2. Следуя инструкциями, отсканировать кубик, чтобы приложение определило его тип.
  3. Подтвердить тип кубика или повторить попытку.
  4. Следуя инструкциям, сканировать каждую сторону кубика.
  5. Проверить выдаваемый результат и подтвердить или повторить попытку.
  6. Сканирование Axis Cube происходит в полу ручном режиме (после сканирования стороны, происходит переход к ручному вводу)

Ручной ввод:

  1. На главной странице нажать кнопку "Ручной ввод"
  2. Закрашивать стикеры кубика, нажимая на цветные кнопки ниже кубика.
  3. Для перехода к следующей стороне нажимать на стрелочки.

Решение:

  1. Выбрать желаемый тип решения
  2. Для перехода между шагами решения нажимать на кнопки "След. шаг" и "Пред. шаг"
  3. Для вывода решения в полном описании нажать на кнопку "Полное описание"
  4. Для вывода решения в языке вращений нажать на кнопку "Язык вращений"
